# Практична робота 2

## Оцінка

Для отримання оцінки 60 - треба виконати:

- 1 завдання 1 рівня.

Для отримання оцінки 75+ - треба виконати:

- 1 завдання 1 рівня
- 1 завдання 2 рівня

Для отримання оцінки 90+ - треба виконати:

- 1 завдання 1 рівня
- 1 завдання 2 рівня
- 1 завдання 3 рівня

Для отримання оцінки 100 - по одному завданню з кожного рівня та їх захистити

Номер завдання обрано відповідно до номеру в журналі.

## Рівень 1

**Прогноз погоди** **(з іконкою):**

- Створіть файл HTML з елементом `<script>`.
- Використовуйте fetch для отримання даних про погоду з API (наприклад, OpenWeatherMap).
- Розберіть JSON-файл з відповіддю API.
Відобразіть поточну температуру, опис погоди та іконку погоди на веб-сторінці.

## Рівень 2

**Генератор** **JSON**
Напишіть програму, яка генерує JSON-файл з заданими даними.

- Використовуйте вбудовані функції мови програмування для створення JSON-структури.
- Збережіть JSON-файл на диск.

## Рівень 3

Створіть веб-додаток для керування бібліотекою книг, де користувачі можуть додавати та видаляти книги. Додайте можливість користувачам робити помітки - вибране, прочитане, бажане і т.д.

## Рівень 4

Розширте свій додаток, додавши форму, де користувачі можуть ввести дані для нового елемента (в новому компоненті).
